# How to setup project locally

Required:

- `node v21.7.3`
- `git`

1. Clone project

```
git clone git@github.com:Umuzi-org/Ezra-Ramatong-280-node-file-io-javascript.git
or
git clone https://github.com/Umuzi-org/Ezra-Ramatong-280-node-file-io-javascript.git
```

2. Install packages

```
npm install
```

3. To run tests

```
npm test
```
